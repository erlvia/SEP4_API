Drivers library and demo for SEP4 Hardware kit
By Erland Larsen, March 2026
VIA UNIVERSITY COLLEGE, Software Technology Engineering
-----------------------------------------------------------
This software is intended for the following hardware setup:
> Arduino Mega2560 board.
> Arduino Uno Multifunktionelt Expansion Shield.
> VIA University expansion board.
> ESP01 + ESP8266 WIFI module.
> HC-SR501 PIR sensor.
> HC-SR04 Proximity sensor (ultra sonic).
> KY-018 LDR module (light sensor).
> DHT11 Air temperature and humidity sensor.
> Capacitive soil moisture sensor.
> SG90 Servo.
-----------------------------------------------------------
See documentation in the DOC folder.
